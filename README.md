## 💫 מהות המצווה

1. **לחזק את רוחו של החולה** – לעודד, לשמח, להזכיר לו שהוא לא לבד.
2. **להתפלל עליו** – כי התפילה נענית ביותר כשנאמרת מתוך נוכחות.
3. **לסייע במעשה** – אם יש צורך באוכל, תרופות, סידורים, סיוע למשפחה.
4. **להאיר באור פנימי** – להכניס למרחב של כאב תחושת אהבה ושמחה.

* **השאר תפילה או פתק ברכה** – משהו מוחשי שממשיך את הביקור גם אחרי.
